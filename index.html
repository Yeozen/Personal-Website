Past Works

These are some of the projects that I've worked on in university, each demonstrating my experience in a variety of domains ranging from AI to sofware development. Feel free to check out my GitHub to see source code for each project.

UltiVision

UltiVision is my final year project, the culmination of 3 years of knowledge and experience, to develop a model using computer vision and deep learning to identify throwing mistakes in ultimate frisbee.

Gen Z Sentiment Analysis

I developed and evaluated 3 different approaches for an intelligent agent to conduct sentiment analysis of Gen Z speech for my Designing Intelligent Agents module. It provided me with experience in Natural Language Processing and Neural Networks.

Ultimate Tournament Manager

Ultimate Tournament Manager is an Android app I wrote using Java built to manage Ultimate Frisbee tournaments but can also be customized to suit different sports.

ForestPoint

ForestPoint was the result my second year software engineering group project, where I worked with 4 other team members to develop a software solution using AI and image processing for a client in the agriculture industry.

CPU Scheduling Algorithms

I worked in a group to implement and evaluate different job scheduling algorithms for the CPU using C programming language for my Operating Systems module in my second year. I gained hands-on experience in C and a deeper understanding the CPU scheduling process.

2048 Game Optimization

In my second year, I worked on a buggy 2048 game and implemented fixes as well as optimizations to the source code according to Object Oriented Programming principles using Java.

Internship Experience

SAINS Sdn. Bhd.

June 2023 - September 2023

During my internship at SAINS, I was assigned to the cybersecurity division and spent my time researching network cybersecurity protocols mainly revolving around OWASP. My main contribution was the development of an automated logging pipeline that could be attached to any existing website and would track and record all HTTP interactions with the website and update a seperate SQLite3 database with the information. This was done using a combination of Python, PHP, SQLite3 and Bash on a Linux VM.
